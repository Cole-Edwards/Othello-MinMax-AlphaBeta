Othello Python:
Basic Python implementation of the Game Othello.

Modes: 
  Player vs Player (PvP)
  Player vs AI (PvAI)
  Trace previous Game

PvP:
  Normal 2 player game.

PvAI:
  Player plays against an AI utilizing a Min-Max algorithm and Alpha-Beta Pruning to decide its moves.

Trace Previous Game:
  Moves are saved and put into a text file to allow the player to go back and retrace their steps.
